# FileShareR Frontend

Frontend application cho hệ thống quản lý và chia sẻ tài liệu - Xây dựng với React, Vite và Tailwind CSS.

## 🎨 Thiết Kế

- **Theme**: Màu xanh nước biển (Ocean Blue) chủ đạo
- **UI Framework**: Tailwind CSS 4
- **Icons**: Lucide React
- **Routing**: React Router v6
- **HTTP Client**: Axios với interceptors

## 🚀 Cài Đặt và Chạy

### 1. Cài đặt dependencies (đã xong)
```bash
npm install
```

### 2. Chạy development server
```bash
npm run dev
```

Ứng dụng sẽ chạy tại: `http://localhost:5173`

### 3. Build for production
```bash
npm run build
```

## 🎯 Tính Năng Đã Hoàn Thành

✅ Authentication (Login, Register, Logout)  
✅ JWT Token Management với Auto Refresh  
✅ Protected Routes  
✅ Dashboard với thống kê  
✅ Responsive Sidebar & Navbar  
✅ Beautiful Ocean Blue Theme  

## 🔗 Backend API

Backend chạy tại: `http://localhost:8080`

Đảm bảo backend đã chạy trước khi start frontend!
